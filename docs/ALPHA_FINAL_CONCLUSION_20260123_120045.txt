ALPHA 최적화 및 Lambda 정규화 종합 분석 최종 결론
생성 시각: 20260123_120045
══════════════════════════════════════════════════════════════════════════════════════

1. 발견한 사실

   1.1 Alpha 불일치의 진짜 원인
       - Grid Search는 Lambda Regularization (λ≈0.01~0.05) 적용
       - ALPHA=1 (baseline)은 정규화 없이 사전 설정
       - 비교가 불공정함 ⚠️

   1.2 TopN별 차이
       - Phase 1 분석: TopN별 최적 alpha가 100% 동일
       - 즉, TopN은 alpha 선택에 영향 없음 ✅

   1.3 Lambda 효과
       - λ 증가 → alpha가 1.0에 가까워짐
       - λ=0: 평균 α=3.86 (Validation 과적합)
       - λ=0.03: 평균 α=1.567843137254902 (안정화)

2. Test 성능 비교 (Fold 1-5 평균)

   ┌─────────────────────────────────────────────────────────┐
   │ ALPHA=1 (Baseline)                                      │
   │   - Test RMSE: nan                     │
   │   - 선택 기준: 사전 설정 (최적화 없음)                 │
   └─────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────┐
   │ 최적 ALPHA (λ=0.030)                              │
   │   - Test RMSE: 1.011298                     │
   │   - 평균 alpha: 1.57                              │
   │   - vs Baseline: +nan%                           │
   └─────────────────────────────────────────────────────────┘

   결과: Baseline이 여전히 더 좋음 (nan% 차이)

3. 근본 원인: Validation Set Overfitting

   3.1 Validation 성능
       - ALPHA=1: Validation MSE = 0.903610
       - 최적 ALPHA: Validation MSE = 0.851077 (5.81% 개선 ✅)

   3.2 Test 성능
       - ALPHA=1: Test RMSE = nan
       - 최적 ALPHA: Test RMSE = 1.011298 (+nan% 악화 ❌)

   3.3 해석
       → Alpha 최적화가 Validation Set에 과적합
       → Lambda 정규화(최대 λ=0.03)로도 해결 불가
       → Validation Set이 Test Set을 대표하지 못함

4. 최종 권장사항

   ┌─────────────────────────────────────────────────────────┐
   │ 실무 사용: ALPHA=1 권장 ⭐                              │
   │   - Test RMSE: nan (최고 성능)         │
   │   - 간단하고 안정적                                     │
   │   - 과적합 위험 없음                                    │
   └─────────────────────────────────────────────────────────┘

   대안 (학술적 엄밀성):
   - Nested Cross-Validation 수행
   - Holdout Set 추가 분할
   - 다른 Validation Set 구성 재검토

5. 공정한 비교 재확인

   Lambda Regularization을 고려한 공정한 비교:
   - 모든 lambda (0.001~0.03)에서 ALPHA=1이 더 우수
   - λ=0 (Pure MSE) 시뮬레이션도 동일 예상
   - 결론: 최적화 자체가 Test 성능 향상에 실패
